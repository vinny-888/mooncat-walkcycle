# mooncat-walkcycle

## Import Module

```
import {MoonCatWalk} from './mooncatWalk.js';
```

## Usage

```
let size = 1;
let tokenId = 0;
let spriteSheet = MoonCatWalk.generateMoonCatSpriteSheet(tokenId, size);
document.getElementById('sprite_sheet').src = spriteSheet;
```
